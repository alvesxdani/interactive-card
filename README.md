# Credit card interactive
Este é um projeto de uma página interativa, onde o usuário coloca seus dados do cartão, onde aparecerá na tela. O projeto é um desafio do Front End Mentor.

# O que foi utilizado:
-   React
-   TypeScript
-   Styled components
-   React-Hook-Form
-   ViteJS
-   React Icons
-   Yup
-   React-Text-Mask
- React Switch
